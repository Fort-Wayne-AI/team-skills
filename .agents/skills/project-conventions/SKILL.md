# Project Conventions

This skill is installed as raw Markdown files at `.agents/skills/project-conventions/`, `.claude/skills/project-conventions/`, and `.hermes/skills/project-conventions/`.

Read it directly from disk. No external service, token, or network access is required.

The REFERENCE.md file in the same directory contains the team's development standards and versioning rules. Follow its instructions when they apply to the current project.
