# Shared Understanding reference

The shared conventions document is maintained in Notion:

- URL: `https://app.notion.com/p/angiecarel/Conventions-39b47eb592be8085a79bf50afd7d6da1?source=copy_link`
- Page ID: `39b47eb592be8085a79bf50afd7d6da1`

The `team-skills read-shared-understanding` command is the supported way to retrieve it. It delegates to the official Notion CLI (`ntn`) and prints Markdown to standard output.

Do not put Notion tokens in this file, project configuration, source control, or chat. Interactive login stores credentials in the local OS credential store; automation can use the locally provided `NOTION_API_TOKEN` environment variable.
